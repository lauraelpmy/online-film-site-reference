# online-film-site-reference
本仓库用于建立一个在线电影与影视站点的参考集合，记录公开可访问站点的基本属性与内容方向。项目目标在于形成稳定、可更新的影视站点信息库，便于长期维护与分析。仓库内容不包含任何影视资源文件，仅整理站点层面的公开信息，不构成推荐或商业宣传。所有记录均基于公开网络信息整理，适合用于资料归档与研究用途。
